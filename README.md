# portal-goldenflowers-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态方式存储，可直接通过浏览器打开查看。

## 项目简介

这是一个纯前端静态页面集合仓库。所有页面均为独立完整 HTML 文件，不依赖后端服务或特定域名。仓库的主要作用是集中保存、版本管理以及方便访问这些页面。

## 目录说明

```
portal-goldenflowers-notes/
├── README.md           # 本文件
├── index.html          # 入口或索引页面（如有）
└── pages/              # 存放所有独立 HTML 页面
    ├── page1.html
    ├── page2.html
    └── ...
```

- `pages/` 目录下存放所有归档的 HTML 页面文件。
- 每个文件均为独立页面，内部包含完整的样式与脚本。
- 根目录下的 `index.html` 可作为默认入口（可选）。

## 页面归档说明

- 所有页面均为静态 HTML 文件，无外部资源依赖（或资源已内联）。
- 页面内容保持原始状态，仅做必要的整理与归档。
- 文件名建议使用英文或拼音，避免特殊字符。
- 每个页面可附带简要说明（如有需要，可在文件内注释）。

## 维护说明

- 新增页面：将 HTML 文件放入 `pages/` 目录，并确保文件完整可打开。
- 更新页面：直接替换对应文件，注意保留原有文件名。
- 删除页面：从 `pages/` 目录移除文件，并检查是否有引用。
- 不建议在本仓库中存放动态内容或需要后端支持的页面。

## 注意事项

- 本仓库仅用于静态 HTML 页面的存储与展示。
- 不涉及任何具体网站、域名或商业用途。
- 页面内容由贡献者自行负责，请确保内容合规。
